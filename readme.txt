== Description ==

This plugin adds a section under My Account page named Product History with a list of all the products the user has bought it.

== Installation ==

To install this plugin just follow the standard procedure, and re-save Permalinks.

or

1. Upload `font-awesome-integration.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/Permalinks and click Save.
